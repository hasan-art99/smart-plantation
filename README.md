# smart-plantation
Aplikasi web PWA untuk Mandor PT Antang Sawit Perkasa guna memantau rotasi panen sawit Divisi C. Fitur input hasil lapangan (HK, JJG, Tonase), rekap otomatis, dan export Excel berwarna. Dukung sistem offline dengan Local Storage di memori HP, sangat ideal untuk area perkebunan yang minim atau tanpa sinyal internet.
